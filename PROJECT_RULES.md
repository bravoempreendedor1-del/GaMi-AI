# GAMI-AI - PROJECT RULES (PURE WEB)

## 1. Identidade
- **Nome:** GaMi-AI.
- **Plataforma:** Web App (Chainlit).
- **Interface:** Focada em Diálogo de Voz e Texto.

## 2. Sistema de Notificação (Novo)
- **Zero Telegram:** Não usar apps externos.
- **Alertas na Tela:** O sistema deve usar recursos visuais (Toasts/Popups) do Chainlit para avisos importantes.
- **Áudio:** O alerta sonoro (fala) é a principal forma de chamar a atenção.

## 3. Comportamento
- **Polímata:** Programador Sênior (Python) e Consultor de Negócios.
- **Voz:** Entrada via Microfone -> Transcrição -> Resposta em Áudio (Auto-play).

## 4. Stack
- Python, Chainlit, OpenAI (Whisper/TTS), LangChain.
